## About this portfolio

This is the source documentation for **Ronald Santana's personal portfolio** — a fast, minimal static site built with [Astro](https://astro.build) and [Bootstrap 5](https://getbootstrap.com). The site showcases projects, skills, and contact information in a clean single-column layout.

## Key features

<CardGroup cols={2}>
  <Card title="Astro-powered" icon="bolt">
    Zero-JS by default. Ships only the HTML and CSS you need for blazing-fast load times.
  </Card>
  <Card title="Bootstrap 5" icon="grid">
    Responsive grid and utility classes out of the box — no custom CSS framework to maintain.
  </Card>
  <Card title="Content Collections" icon="database">
    Projects are managed as Markdown files with type-safe frontmatter validated by Zod.
  </Card>
  <Card title="Smooth animations" icon="wand-magic-sparkles">
    CSS keyframe animations on load and Intersection Observer for scroll-triggered effects.
  </Card>
</CardGroup>
